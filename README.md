# NUSSIF

NUS Student Investment Fund — A student-led platform at NUS building Asia's premier live student fund.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm run test
```

## Tech Stack

- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- React Router
- Vitest + Testing Library
